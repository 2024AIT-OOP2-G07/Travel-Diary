# Travel_Diary
旅行した場所と評価をマップ上に表示する

## セットアップ方法
### リポジトリのクローン
```shell
git clone https://github.com/2024AIT-OOP2-G07/Travel-Diary
```

### パッケージのインストール
```shell
pip3 install -r requirements.txt
```

## 実行方法
```shell
python3 app.py
```

## シード
デモ用のデータを追加する
※既存のデータは削除されます
```shell
python3 seed.py
```
